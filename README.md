# Portfolio Project — Step 1 Setup

- **Tools I installed:** Cursor IDE (v3.9.16), Claude Code extension (`anthropic.claude-code` v2.1.195), Codex extension (`openai.chatgpt` v26.623.31443), Git, and GitHub CLI (`gh`).

- **Steps I completed:** Installed and configured the required Cursor extensions, opened the project workspace, authenticated Claude Code and Codex, initialized the repository, and published it to GitHub at https://github.com/proongt/proong-portfolio

- **Issues I ran into and how I solved them:** Cursor extension shortcuts required an active workspace, resolved by opening the project directory from the editor. Codex was accessed via the command palette once the workspace was loaded. GitHub deployment required re-authenticating through `gh auth login`, after which the repository was created and pushed successfully with `gh repo create`.
